# 3D Name (Next.js + react-three-fiber)

- Stack: Next.js App Router (TS), @react-three/fiber, @react-three/drei, Tailwind.
- Scene: Floating 3D text with soft shadows and orbit controls.

Quick start

1. Copy .env.local.example to .env.local and set your display name.
2. Run the dev server.

Config

- NEXT_PUBLIC_DISPLAY_NAME controls the text shown.
